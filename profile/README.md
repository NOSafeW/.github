# NoSafeW

NoSafeW 是一个面向中文用户的公开安全资料项目，整理 SafeW / SafeX 相关公开报告、应用包名、截图证据、时间线与自救指南。

我们的目标很简单：当普通用户在安装 SafeW、SafeW Messenger、SafeW 小飞机、SafeX 聊天之前搜索相关信息时，能先看到可核对的安全风险，而不是只看到推广站点和自证清白的软文。

## 重点仓库

| Repository | 内容 |
| --- | --- |
| `evidence` | SafeW / SafeX 公开证据、包名、截图、PDF 报告与来源链接 |
| `timeline` | 2025-2026 年公开记录时间线 |
| `user-safety-guide` | 面向中文用户的卸载、自查、助记词风险处理指南 |
| `brand-and-domains` | SafeW / SafeX 名称、包名、域名与搜索矩阵记录 |
| `media-kit` | Telegram、X、Reddit、博客与短视频传播素材 |

## 核心页面

## 写作原则

- 以公开来源、截图、包名、hash 和可核对时间线为基础。
- 区分“已验证事实”和“基于公开材料的推断”。
- 避免私人信息、骚扰、威胁和无法核对的指控。
- 中文优先，降低非技术用户理解门槛。
